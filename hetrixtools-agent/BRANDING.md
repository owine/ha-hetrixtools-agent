# Branding assets needed

ImageMagick was not available when this repository was set up, so the branding placeholder images were not generated automatically.

Before publishing this app, add the following files to this directory:

- `icon.png` — 256×256 pixels, PNG format. Used as the app icon in the HA app store.
- `logo.png` — 512×256 pixels (or similar wide aspect ratio), PNG format. Used as the header logo on the app's detail page.

Both images should represent the HetrixTools brand. Refer to the [Home Assistant add-on documentation](https://developers.home-assistant.io/docs/add-ons/presentation) for any current requirements on image format or dimensions.
