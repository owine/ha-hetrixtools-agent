# ha-hetrixtools-agent

Home Assistant repository containing the **HetrixTools Agent** app (formerly called an add-on). The app runs the HetrixTools server monitoring agent on your HA OS host, reporting CPU, RAM, disk, network, and other metrics to your HetrixTools dashboard.

## Install

Add this repository to your Home Assistant app store:

1. **Settings → Apps → app store** (shopping bag icon, top right)
2. Three-dot menu → **Repositories**
3. Add: `https://github.com/owine/ha-hetrixtools-agent`
4. Install **HetrixTools Agent**, set your Server ID, and start the app.

For complete documentation including all configuration options and HA OS caveats, see [hetrixtools-agent/DOCS.md](hetrixtools-agent/DOCS.md).
